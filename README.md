## 介绍

lmo-Lrc-Editor，一款开源的歌词文件制作系统。

## 内置功能

- 选择音频文件
- 选择lrc文件
- 导出lrc文件
- 编辑时间点
- 预览



## 在线体验

[演示地址](https://lrc.ayuanlmo.cn)

## 开发
```bash
#clone
git clone https://github.com/ayuanlmo/lmo-lrc-editor
#安装模块
yarn
#启动项目
yarn dev
```

## 发布

```bash
yarn build
```
